# Lofty Sunset Scroll Concept

A production-ready React + Vite + GSAP ScrollTrigger cinematic landing-page concept for Lofty Coffee Roasting Works.

## Run locally

```bash
npm install
npm run dev -- --port 5174
```

## Build

```bash
npm run build
```

## Deploy

Push to GitHub, then import into Vercel. Vercel will use `vercel.json` and output the `dist` directory.
